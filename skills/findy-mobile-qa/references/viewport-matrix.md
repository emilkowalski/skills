## 화면별 시나리오

### 인증

- 입력 포커스 후 다음 필드와 CTA로 이동 가능
- 키보드가 약관과 가입 버튼을 가리지 않음
- 오류 메시지가 입력값을 지우지 않음
- 뒤로가기가 현재 단계만 한 단계 닫음

### 홈과 검색

- 2x3 분야 카드가 같은 크기와 간격을 유지
- 검색창, 추천 콘텐츠, 커뮤니티, 푸터까지 스크롤 가능
- 결과 없음, 오프라인, 시간 초과 상태에서 복구 가능

### 커뮤니티와 상세

- 모든 탭과 필터가 도달 가능
- 카드 본문, 미디어, 반응 영역이 잘리지 않음
- 댓글 입력 시 현재 댓글 맥락과 보내기 버튼이 보임
- 대댓글 작성 취소와 제출이 명확함

### 스냅과 비디오

- 분야·정렬·보기 전환이 충돌하지 않음
- 1열/3열 전환 후 현재 페이지와 스크롤이 예측 가능
- 영상 로딩, 음소거, 소리, 재개, 탭 이탈 정지를 확인

### 프로필과 설정

- 프로필 태그와 그리드가 하단 내비게이션에 가리지 않음
- 설정 마지막 항목까지 완전히 노출
- 스크롤 중 헤더 로고와 화면명이 크롭되지 않음
- 저장 토스트가 하단 내비게이션 위에 표시

## 결함 기록

```text
화면:
환경 ID:
글자 배율:
키보드/권한/네트워크 상태:
재현 절차:
기대 결과:
실제 결과:
심각도:
자동 테스트 여부:
실기기 확인 여부:
```

스크린샷 비교는 같은 크기, 같은 스크롤 위치, 같은 데이터로 만든다. 사용자가 요청하지 않으면 파일 경로와 결과만 보고하고 이미지를 메시지에 첨부하지 않는다.
